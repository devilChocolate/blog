    # Dom事件流的概念

    • 在页面上点击元素产生事件,事件会在元素到根节点之间沿着特定顺序传播,沿途的节点都会收到该事件，这个传播过程称为DOM事件流

    # 事件冒泡概念

    • 元素产生事件后,沿着元素传播到根节点(由内向外)。在冒泡的过程中，最事件源的事件先被触发，然后依次向外执行，直到触发最外面的元素

    # 事件冒泡流程图

    # 事件冒泡实例

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <style>
      #div1{
        padding: 0;
        margin: 0;
        width: 200px;
        height: 200px;
        background: yellow;
      }
      #div2{
        padding: 0;
        margin: 0;
        width: 100px;
        height: 100px;
        background: blue;
      }
      #div3{
        padding: 0;
        margin: 0;
        width: 50px;
        height: 50px;
        background: green;
      }
    </style>
    <div id="div1">
      <div id="div2">
        <div id="div3"></div>
      </div>
    </div>
    <script>
      const div1 = document.getElementById("div1")
      div1.onclick = function () {
        console.log("div1");
      }
      const div2 = document.getElementById("div2")
      div2.onclick = function () {
        console.log("div2");
      }
      const div3 = document.getElementById("div3")
      div3.onclick = function () {
        console.log("div3");
      }
    </script>
  </body>
</html>

    # 阻止事件冒泡

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

  </head>

      #div1{
        padding: 0;
        margin: 0;
        width: 200px;

        background: yellow;
      }
      #div2{
        padding: 0;
        margin: 0;
        width: 100px;
        height: 100px;
        background: blue;
      }
      #div3{
        padding: 0;
        margin: 0;
        width: 50px;
        height: 50px;
        background: green;
      }
    </style>
    <div id="div1">
      <div id="div2">
        <div id="div3"></div>
      </div>
    </div>
    <script>
      const div1 = document.getElementById("div1")
      div1.onclick = function () {
        console.log("div1");
      }
      const div2 = document.getElementById("div2")
      div2.onclick = function () {
        console.log("div2");
      }
      const div3 = document.getElementById("div3")
      div3.onclick = function (event) {
        //使用stopPropagation阻止事件冒泡
        event.stopPropagation()
        console.log("div3");
      }
    </script>
  </body>
</html>

    # 事件捕获概念

    • 元素产生事件后,最外层节点捕获事件,沿着根节点到产生事件的元素(由外向内),在捕获的过程中，最外层（根）元素的事件先被触发，然后依次向内执行，直到触发最里面的元素（事件源）

    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

  </head>
  <body>
    <style>
      #div1{
        padding: 0;
        margin: 0;
        width: 200px;
        height: 200px;
        background: yellow;
      }
      #div2{
        padding: 0;
        margin: 0;
        width: 100px;
        height: 100px;
        background: blue;
      }
      #div3{
        padding: 0;
        margin: 0;
        width: 50px;
        height: 50px;
        background: green;
      }
    </style>
    <div id="div1">
      <div id="div2">
        <div id="div3"></div>
      </div>
    </div>
    <script>
      const div1 = document.getElementById("div1")
      div1.addEventListener('click',function () {
        console.log("div1");

        event.stopImmediatePropagation()
      },true)
      const div2 = document.getElementById("div2")
      div2.addEventListener('click',function () {
        console.log("div2");
      },true)
      const div3 = document.getElementById("div3")
      div3.addEventListener('click',function () {
        console.log("div3");
      },true)
    </script>
  </body>
</html>
