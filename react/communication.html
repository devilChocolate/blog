    组件的通信方式

    wangkai 2022/3/10

    # 父传子

    • 父组件通过props传递数据,子组件中通过this.props接受传递过来的数据
      <A name="张三" age={18} address="瓦罗兰大陆"/>
  //简写 通过{...data} 传递数据
  const o = {
    name: '张三',
    age: 18,
    address: '瓦罗兰大陆'
  }
  <A {...o}/>
  --------------------
  //A组件中 使用this.props.name ...获取传递过来的数据

    # 子传父

    • 父组件通过props传递一个函数,子组件中通过调用该函数并且传递参数的形式向父组件中传递数据
      import A from './A'
  class B extends Components{
    handleUpdateName = (info)=>{
      console.log(`执行了handleUpdateName,子组件传递的信息为${info}`,)
    }
    render(){
      return(
         <A handleUpdateName = {this.handleUpdateName}/>
      )
    }
  }
  --------------------
  class A extends Components{
    handleClick = ()=>{
      const info = "这是子组件向父组件传递的信息"
      //通过this.props 获取父组件传递的函数 并且调用该函数 传递信息
      this.props.handleUpdateName(info)
    }
    render(){
      return(
         <button onClick={this.handleClick}></button>
      )
    }
  }
